# 台灣高鐵 PDF 收據自動解密工具 (THSRC Receipt Decryptor)

這是一個專為台灣高鐵 (THSRC) 電子收據設計的自動解密工具。由於高鐵收據 PDF 通常使用購票當日的日期（格式為 `YYYYMMDD`）作為密碼，本工具透過自動嘗試指定日期範圍內的密碼，快速移除 PDF 的密碼保護並另存為無密碼版本。

## 功能特點

- **自動化密碼嘗試**：自動產生過去 N 天內的日期字串作為密碼進行比對。
- **高效能並行處理**：同時開啟 4 個執行緒進行解密，大幅縮短等待時間。
- **跨平台支援**：支援 Linux (x86_64) 與 macOS (Apple Silicon)。
- **智慧檔名處理**：自動處理重複檔名，確保解密後的檔案不會互相覆蓋。

## 編譯方式

本專案提供 `Makefile` 方便快速編譯。請確保您的系統已安裝 Go 環境。

```bash
# 編譯所有平台版本
make build

# 或是單獨編譯特定版本
make linux-x86
make mac-arm
```

編譯完成後的執行檔將存放於 `bin/` 目錄中。

## 使用方法

執行編譯後的程式，並透過參數指定輸入資料夾與日期範圍：

```bash
./bin/thsrc-decrypt-darwin-arm64 --input-folder ./my_receipts --date-range 60d
```

### 參數說明

- `--input-folder`: 放置原始加密 PDF 檔案的資料夾路徑（預設為 `./input_pdfs`）。
- `--date-range`: 密碼嘗試的日期範圍，例如 `30d` 表示嘗試從今天起往回推 30 天的日期（預設為 `60d`）。
- `-h, --help`: 顯示幫助訊息。

## 運作原理

1. 程式會掃描 `--input-folder` 中所有的 `.pdf` 檔案。
2. 根據 `--date-range` 產生一系列 `YYYYMMDD` 格式的日期字串。
3. 使用多執行緒嘗試對每個 PDF 檔案進行解密。
4. 解密成功後，會將移除密碼後的 PDF 存放在 `./decrypted` 資料夾中。

## 注意事項

- 本工具僅適用於以「日期」作為密碼的高鐵收據。
- 輸出的 PDF 將不再受密碼保護，請妥善保管。
